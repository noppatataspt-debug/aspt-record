# ASPT Production System

ระบบบันทึกและแสดงผลข้อมูลการผลิตของโรงงาน ASPT

## URLs
- **บันทึกข้อมูล (PWA):** https://aspt-record.vercel.app
- **Dashboard:** https://aspt-record.vercel.app/dashboard.html

## Tech Stack
- HTML/CSS/JS (Vanilla)
- Supabase (PostgreSQL + REST API)
- Vercel (Hosting)

## Files
- `index.html` — PWA สำหรับบันทึกข้อมูลการผลิต
- `dashboard.html` — Dashboard แสดงกราฟและ KPI
- `manifest.json` — PWA config
- `vercel.json` — Vercel routing config
